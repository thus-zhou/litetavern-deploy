# 📱 远程访问指南 (Remote Access Guide)

本指南将教你如何在**手机**或**外网**环境下访问运行在电脑上的 LiteTavern。

---

## 方案一：一键穿透 (推荐新手)
无需注册账号，无需配置，直接运行脚本即可。使用的是 `localtunnel` 服务。

### 步骤：
1. 确保主程序正在运行（即 `python run.py` 窗口未关闭）。
2. 双击项目根目录下的 **`share_app.bat`** 脚本。
3. 脚本会自动获取一个**验证密码**（通常是你的 IP 地址）并显示在屏幕上。
4. 脚本会生成一个类似 `https://xxxx-xxxx.loca.lt` 的网址。
5. 在手机浏览器打开该网址。
6. **重要**：如果出现 "Friendly Reminder" 页面，请在 "Tunnel Password" 框中输入脚本显示的密码，点击 "Click to Submit"。

**注意**：此服务由第三方免费提供，可能会有不稳定的情况。

---

## 方案二：使用 Cpolar (推荐国内用户)
如果你觉得方案一速度慢或不稳定，推荐使用 Cpolar（国内访问速度快）。

### 步骤：
1. 访问 [Cpolar 官网](https://www.cpolar.com/) 注册账号并下载 Windows 客户端。
2. 安装并登录 Cpolar。
3. 打开 Cpolar 终端，输入命令映射端口 **8000**：
   ```bash
   cpolar http 8000
   ```
4. 复制生成的公网链接（如 `http://xxxx.cpolar.cn`）到手机打开即可。

---

## 方案三：局域网访问 (无需流量)
如果你的手机和电脑连接的是**同一个 WiFi**。

1. 在电脑上按下 `Win + R`，输入 `cmd` 回车。
2. 输入 `ipconfig`，找到 "IPv4 地址" (例如 `192.168.1.5`)。
3. 手机浏览器直接访问：`http://192.168.1.5:8000` (注意把 IP 换成你查到的)。
